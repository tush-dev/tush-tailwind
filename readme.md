# 🚀 Tush Tailwind – Mini Tailwind CSS Clone

A lightweight **utility-first styling engine** built using **vanilla JavaScript**.  
This project dynamically applies styles based on class names — just like Tailwind CSS.

---

## ✨ Features

### 🎨 Text Colors
- `tush-red`
- `tush-blue`
- `tush-green`
- `tush-yellow`
- `tush-violet`

### 🖼 Background Colors
- `tush-bg-red`
- `tush-bg-blue`
- `tush-bg-green`
- `tush-bg-yellow`
- `tush-bg-violet`

### 📦 Spacing
- `tush-p-10` → padding: 10px  
- `tush-m-20` → margin: 20px  

### 📏 Dimensions
- `tush-w-100` → width: 100px  
- `tush-h-200` → height: 200px  

### 📐 Flexbox
- `tush-flex` → display: flex  
- `tush-center` → center content (horizontal + vertical)
